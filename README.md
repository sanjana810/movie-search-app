This project is a Movie Search Application that allows users to search for movies and view details such as title, year, and poster in real time. The application integrates 
with the OMDb API to fetch movie data dynamically and displays results in a responsive card layout.

Features:
- Search movies by name
- Display multiple movie results
- Movie posters and release year
- Error handling for invalid searches
- Responsive user interface

Technologies Used:
- HTML
- CSS
- JavaScript
- REST API
